# 核心技术

提示

版本 5.2.6.RELEASE

参考文档的这一部分包含了 Spring Framework 所有必须的技术。

其中最重要的是Spring框架的控制反转（IoC）容器。 对Spring框架的IoC容器深入讨论之后，将全面介绍Spring的面向方面编程（AOP）技术。 Spring框架拥有自己的AOP框架，该框架在概念上易于理解，并且成功解决了Java企业编程中80％的AOP需求的难题。

还提供了Spring与AspectJ的集成（目前是功能最丰富的，当然也是Java企业领域中最成熟的AOP实现）。
